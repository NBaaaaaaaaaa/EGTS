# Как запустить проект?
## Шаг 1
В файле *main_db/m_config.py* изменить значения переменных на свои.
```
host = ""
user = ""
password = ""
```

Эти значения рекомендуем оставить по умолчанию.
```
db_name = "egts_db"
table_name = "egts_data"
```

## Шаг 2
Запустить файл *main_db/create_db.py*.

## Шаг 3

## Шаг 4
